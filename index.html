Projects

Merch Perch

3rd Place, StubHub Boston Hackathon, May 2013

Created over a 36 hour span for the inaugural StubHub Boston Hackathon, with the goal being to "transform the fan experience." MerchPerch is a merchandise sales platform. Vendors catalog their merchandise for upcoming events through the vendor portal. Customers can then browse merchandise before or during the show, pay with Stripe, and pick up their merchandise with their receipt on their smartphone.

SPIMS

Semester Project Image Matching Software

Created for Northeastern's Software Development course, SPIMS is a program which searches for a set of given pattern images within the set of given source images. We achieved this through normalized cross-correlation, using Fourier transforms to speed up the process. Written in Python, making liberal use of SciPy and NumPy.
